# Message System
---
프로세스 사이에 공유 변수(shared variable)를 일체 사용하지 않고 통신하는 시스템

# Direct Communication
---
통신하려는 프로세스의 이름을 명시적으로 표시. 커널을 통해서 전달 함

# Indirect Communication
---
mailbox (또는 port)를 통해 메시지를 간접 전달. 커널을 통해서 전달 함
