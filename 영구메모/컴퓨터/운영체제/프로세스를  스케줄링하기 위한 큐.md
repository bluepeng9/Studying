# 개념
---
- Job queue
	- 현재 시스템 내에 있는 모든 프로세스의 집합
- Ready queue
	- 현재 메모리 내에 있으면서 CPU 를 잡아서 실행되기를 기다리는 프로세스의 집합
- Device queue
- I/O 디바이스의 처리를 기다리는 프로세스의 집합

프로세스들은 각 큐들을 오가며 수행된다.
