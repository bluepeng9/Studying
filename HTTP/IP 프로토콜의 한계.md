# 비연결성

- 패킷을 받을 대상이 없거나 서비스 불능 상태여도 패킷을 전송할 수 있습니다.
	- 예) 친구가 이사를 갔어도 편지를 보내는 경우
- 클라이언트는 대상 서버가 패킷을 받을 수 있는 상태인지 모릅니다.

---
# 비신뢰성 
- 중간에 패킷이 사라질 수 있습니다.
	- ex) 중간 노드가 꺼져 전송이 안 된 경우
- 패킷이 순서대로 안 올 수 있습니다.
	- ex) 메시지를 분리해서 보낼 때(MTU) "Hello world"가 "world Hello"로 순서가 바뀌어 도착할 수 있습니다.
